# 🍳 Recetario
Una aplicación web progresiva (PWA) *Offline-First* diseñada para resolver el problema diario de qué cocinar con los ingredientes que ya tienes en casa. Optimizada para ofrecer recetas rápidas, caseras y tradicionales mexicanas en menos de 30 minutos.

## ✨ Características Principales

* **Búsqueda Estricta (Lógica AND):** Un motor de filtrado de precisión milimétrica que cruza los ingredientes de tu alacena con una base de datos local para mostrar solo los platillos que realmente puedes preparar.
* **Offline-First:** Al ser una PWA con almacenamiento estático en JSON, la aplicación completa se descarga en el dispositivo y funciona a la velocidad de la luz, sin requerir conexión a internet.
* **Base de Datos Masiva:** Incluye un catálogo curado de 347 recetas tradicionales (tacos, sopes, guisados, etc.) estructuradas en formato JSON.
* **Diseño Minimalista e Intuitivo:** Interfaz construida con principios de *Glassmorphism* (efecto cristal esmerilado) y tipografía moderna para una lectura clara en dispositivos móviles.

## 🛠️ Stack Tecnológico

* **Frontend:** HTML5, Vanilla JavaScript.
* **Estilos:** Tailwind CSS (vía CDN) para un diseño responsivo y utilitario.
* **Base de Datos:** Estructura de datos JSON local.
* **Arquitectura:** Single Page Application (SPA) / Progressive Web App (PWA).
